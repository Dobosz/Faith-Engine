Instructions for including and linking 
-------------------------------------- 
These are the following parameters you should append to your project linker settings.  
  
* Additional Include Directories (.h) C:\NxOgre-GIT\sdk  
* Additional Library Directories (.lib) C:\NxOgre-GIT\sdk  
* Additional Dependencies - NxOgre.lib 
 
Then copy the file C:\NxOgre-GIT\sdk\NxOgre.dll to your application directory.  
 
For a first time user, you should follow one of the many tutorials at http://nxogre.org or have a look at Cake to set up your application.
